## Git 명령어



#### 1. 명령어

---------------------

1. **Pwd**
   - 현재 폴더의 경로
2. **ls**
   - 현재 폴더 표시
   - ls - al : 세부표시
3. **cd 폴더명**
   - 해당 폴더로 이동
4. **cd .. **
   - 상위 폴더로 이동
5. **mkdir [폴더명]**
   - 폴더 생성
6.  **rmdir [폴더명]**, **rm [파일명]**
   - 폴더 삭제 또는 파일 삭제
7.  ** mv [파일 및 폴더명] ** 
   - 파일 이동
8. **touch [파일명] **
   - 파일을 생성 (확장자 명까지)
9. echo
   - 파일 출력
